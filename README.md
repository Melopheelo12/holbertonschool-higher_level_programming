# holbertonschool-higher_level_programming

# 🧠 Higher Level Programming

This repository contains all the projects completed as part of the **Holberton School Full-Stack Software Engineering program**.

The focus of this repository is on **high-level programming languages**, mainly:
- Python 🐍
- JavaScript ⚡
- SQL 🗄️

---

## 📚 Project Overview

This repository is organized into multiple directories, each covering a specific concept or technology:

### 🐍 Python
- `0x00-python-hello_world` → Introduction to Python
- `0x01-python-if_else_loops_functions` → Conditions, loops, and functions
- `0x02-python-import_modules` → Modules and imports
- `0x03-python-data_structures` → Lists, tuples, and data structures
- `0x04-python-more_data_structures` → Sets, dictionaries
- `0x05-python-exceptions` → Error handling
- `0x06-python-classes` → Object-Oriented Programming
- `0x07-python-test_driven_development` → TDD approach
- `0x08-python-more_classes` → Advanced OOP
- `0x09-python-everything_is_object` → Python object model
- `0x0A-python-inheritance` → Inheritance
- `0x0B-python-input_output` → File handling
- `0x0C-python-almost_a_circle` → Full project (OOP + tests)

### 🗄️ SQL
- `0x0D-SQL_introduction` → Basic SQL queries
- `0x0E-SQL_more_queries` → Advanced queries

### 🔗 Python + SQL (ORM)
- `0x0F-python-object_relational_mapping`

### 🌐 Networking
- `0x10-python-network_0`
- `0x11-python-network_1`

### ⚡ JavaScript
- `0x12-javascript-warm_up`
- `0x13-javascript_objects_scopes_closures`
- `0x14-javascript-web_scraping`
- `0x15-javascript-web_jquery`
